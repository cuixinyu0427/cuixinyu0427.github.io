部署到GitHub

注册github账号

直接注册即可:github

新建username.github.io仓库

注册成功后，在github首页单击头像->Your repositories
在自己的 GitHub 账号下创建一个新的仓库，命名为 username.github.io（username是你的账号名)。

配置Git与GitHub

打开git bash,设置用户名称和邮件地址

设置完成后为了能够在本地使用 git 管理 github 上的项目，需要绑定SSHkey。

这一步骤推荐在git bash中运行指令。若使用powershell，less语句缺少必要C语言环境，需要访问
C:\Users\Username.ssh\id_rsa.pub复制
打开github网页
单击头像->settings,在设置页面找到SSH and GPG keys
单击New SSH key
